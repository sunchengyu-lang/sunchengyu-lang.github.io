Hello Hexo

Hello World

以下是一些 Hexo 常用指令的说明：

  1. hexo new "文章标题"：创建新文章。使用这个指令可以在 Hexo 项目中创建一篇新的文章，替换 “文章标题” 为你实际的文章标题。

  2. hexo generate：生成静态文件。这个指令会将 Markdown 文件转换为 HTML 文件，生成静态的博客网站。

  3. hexo server：启动本地服务器。使用这个指令可以在本地预览生成的博客网站，通过浏览器访问 http://localhost:4000 来查看。

  4. hexo deploy：部署到线上服务器。这个指令用于将生成的静态文件部署到线上服务器，例如将博客发布到 GitHub Pages 或其他托管平台。

  5. hexo clean：清理缓存。这个指令会清理 Hexo 项目中的缓存文件，可以在出现问题时尝试使用。

  6. hexo new_page "页面名称"：创建新页面。使用这个指令可以创建一个新的页面，页面名称替换为你想要创建的实际页面名称。

  7. hexo new_draft "草稿标题"：创建草稿。这个指令用于创建一篇草稿文章，草稿文章不会被生成为静态文件，只在本地可见，可以在日后继续编辑和发布。

  8. hexo publish "草稿标题"：发布草稿。使用这个指令可以将草稿文章发布为正式文章，使其可见并生成为静态文件。
